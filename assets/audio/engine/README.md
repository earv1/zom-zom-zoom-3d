# Engine Audio

## Folder Structure

Place files here following this naming convention:

### Engine loops (one per gear)
gear_1.ogg
gear_2.ogg
gear_3.ogg
gear_4.ogg
gear_5.ogg

### Gear shift
shift.ogg

## Requirements
- All engine loops must be seamlessly loopable
- OGG format preferred (smaller, Godot handles well)
- Loops should be recorded at a consistent RPM for that gear band
- Shift sound should be short — under 0.3s ideally
