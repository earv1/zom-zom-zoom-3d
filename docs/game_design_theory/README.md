## Key Insight

**Fun and addictive are not the same thing.**

The most enduringly beloved games (Chess, Tetris, Hades, Slay the Spire) are not addictive in a manipulative sense — they are deeply *learnable*, provide *genuine mastery*, satisfy real psychological needs, and layer goals across multiple timescales so players always have something meaningful to pursue. Compulsion-loop design can drive short-term retention but kills long-term love.

Design for **eudaimonic** enjoyment (meaning, mastery, growth) first; hedonic pleasure (moment-to-moment fun) follows naturally from good mechanics.
